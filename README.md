# Network Traffic Analysis using Wireshark

## Overview
This project focuses on analyzing ARP requests and network traffic behavior using Wireshark. The objective was to study IP-MAC address mapping, detect network anomalies, and understand ARP spoofing concepts.

## Features
- ARP packet analysis
- Network traffic monitoring
- Detection of suspicious ARP activity
- Study of ARP spoofing behavior
- Packet filtering using Wireshark

## Tools Used
- Wireshark
- Networking Fundamentals

## Key Learnings
- Packet capturing and filtering
- ARP request/reply analysis
- Network troubleshooting concepts
- Basic cybersecurity monitoring techniques
